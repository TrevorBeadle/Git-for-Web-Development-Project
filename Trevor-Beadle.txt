    1. What is Semantic HTML? 
    -The use of HTML in a way that focuses on meaning rather than appearance.

    2. What is HTML used for? 
    -HTML is a markup language used for web development. This is the language that the browser actually reads.

    3. What is an attribute and where do we put it? 
    -Attributes are a way to supply data to an HTML element without it displaying on the page.

    4. What is the h1 tag used for? How many times should I use it on a page? 
    -h1 is the largest heading tag and it should only be used once on the page.

    5. Name two tags that have required attributes. 
    <a> and <img>

    6. What do we put in the head of our HTML document? 
    -Metadata, such as the <title>, <style>, etc.

    7. What is an id? 
    -Titles that only appear on one element.

    8. What elements can I add an id to? 
    -They can be used on any HTML element.

    9. How many times can I use the same id on a page? 
    -Once

    10. What is a class? 
    -A title that can appear on multiple elements.

    11. What elements can I add a class to? 
    -Also on any HTML element.

    12. How many times can I use the same class on a page? 
    -As many times as necessary.

    13. How do I get my link to open in a new tab? 
    <a href="website.com" target="_blank">website.com</a>

    14. What is the alt attribute in the image tag used for? 
    -Specifying alternate text if the image cannot be displayed. Used mostly for screen readers.

    15. How do I reference an id?  
    #idName

    16. What is the difference between a section and a div 
    -Section is semantic, div is not.

    17. What is CSS used for? 
    -Styling your HTML.

    18. How to we select an element? Example - every h2 on the page

    19. What is the difference between a class and an id? - Give me an example of when I might use each one 
    -Ids can only be used on one element, classes can be used on multiple. You would use an id to add styling to a specific picture, and only that picture will be styled that way. A class would be used to style multiple pictures the same way.

    22. What is a parent child selector? When would this be useful? 
    A CSS selector that allows you to select all of the child elements of a parent element. It is useful when needing to style all of the list items of a list.

    23. How do you select all links within a div with the class of sidebar?

    24. What is a pseudo selector?
    -A selector that selects elements that are in a specific state, such as hover selects elements when the mouse hovers over that element.

Stretch Questions

    1. What is the difference between an inline element and a block element?
    -Inline elements do not start on a new line and only take up as much width as necessary. Block elements start on a new line and take up the entire width of the page.

    2. What happens when an element is positioned absolutely? 
    It is removed from the normal document flow, and no space is created for the element in the page layout. It is positioned relative to its closest positioned ancestor.

    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
    You can use the inline-block property.

    4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
    Block: div, address, article
    Inline: a, button
    Inline-block: nav

    5. In your own words, explain the box model. What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them?
    All HTML elements are considered boxes, and the box model outlines how padding, border, and margin surround the content of each element.
    The fix for the box model is box-sizing:border-box which includes padding and borders in an element's width.
